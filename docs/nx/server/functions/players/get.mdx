# Get

This function well create a new vehicle and return it in the cb.

```typescript
NX.Players.Get(source)
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript">

```typescript
const NX = exports.NX.useServer()

const nxPlayer = NX.Players.Get(1)

nxPlayer.AddItem('bread', 3)
```

</TabItem>
