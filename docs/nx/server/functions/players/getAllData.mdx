# GetData

This function well get all players and return all data.

```typescript
NX.Players.GetAllData()
```

:::danger
Don't edit the data that is returned.
:::

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript/JavaScript">

```typescript title="/rc/server/player/player.service.ts"
const NX = exports.NX.useServer()

const nxPlayers = NX.Players.GetAllData()

for (const nxPlayer of nxPlayers) {
  console.log(nxPlayer.uid)
}
```

</TabItem>
<TabItem value="lua" label="Lua">

```lua title="/rc/server/player/player.service.ts"
local NX = exports['NX']:useServer()

local nxPlayers = NX.Players.GetAllData()

for key in pairs(nxPlayers) do
    print(nxPlayer.uid)
end
```

</TabItem>
</Tabs>

---

## Returns

```js
{
    accounts,
    position,
    identifier,
    inventory,
    charinfo,
    permissions,
    weight,
    source,
    uid,
    maxWeight,
    name,
}
```
