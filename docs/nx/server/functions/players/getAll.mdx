# GetAll

This function well return the sources of all connected players.

```typescript
NX.Players.GetAll()
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript">

```typescript title="/rc/server/player/player.service.ts"
const NX = exports.NX.useServer()

const nxPlayers = NX.Players.GetAll()

for (const nxPlayer of nxPlayers) {
  nxPlayer.AddItem('bread', 1)
}
```
