# OnServerEvent

This function well listen for server side events.

```typescript
NX.Misc.OnServerEvent(eventName, cb)
```

---

## Example

:::caution
**last callback** argument is always the callback to respond to the client side
:::

<Tabs>
<TabItem value="ts" label="TypeScript/JavaScript">

```typescript title="/rc/server/events/index.ts"
const NX = exports.NX.useServer()

NX.Misc.OnServerEvent('NX::helloWorld', (arg1, arg2, cb) => {
  console.log(arg1, arg2)
  cb('Hello world')
})
```

</TabItem>
<TabItem value="lua" label="Lua">

```lua title="/rc/server/events/index.ts"
local NX = exports['NX']:useServer()

NX.Misc.OnServerEvent("NX::helloWorld", function(arg1, arg2, cb)
    print(arg1, arg2)
    cb('Hello world')
end)
```

</TabItem>
</Tabs>

## Arguments

| Argument  | Data Type | Optional | Default Value | Explanation                                                  |
| --------- | --------- | -------- | ------------- | ------------------------------------------------------------ |
| eventName | String    | No       | -             | The event name to listen to                                  |
| cb        | Function  | No       | -             | The callback that well be executed after receiving the event |
