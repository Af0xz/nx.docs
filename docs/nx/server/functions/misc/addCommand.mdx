# AddCommand

This function well create a new command.

```typescript
NX.Misc.AddCommand(name, cb, authPermissions)
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript">

```typescript title="/rc/server/commands/commands.servcice.ts"
const NX = exports.NX.useServer()

NX.Misc.AddCommand(
  'kill',
  (source, args) => {
    //
  },
  ['admin']
)
```

## Arguments

| Argument        | Data Type | Optional | Default Value | Explanation                                                |
| --------------- | --------- | -------- | ------------- | ---------------------------------------------------------- |
| name            | String    | No       | -             | The command name                                           |
| cb              | Function  | No       | -             | The callback that well be executed after command execution |
| authPermissions | Array     | yes      | ['user']      | The required permissions to execute the command.           |
