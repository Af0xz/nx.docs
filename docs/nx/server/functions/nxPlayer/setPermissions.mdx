# SetPermissions

:::info
This function will set the player permissions
:::

```typescript
nxPlayer.SetPermissions(permission)
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript/JavaScript">

```typescript title="/rc/server/player/player.class.ts"
const NX = exports.NX.useServer()

const nxPlayer = NX.Players.Get(source)

nxPlayer.SetPermissions('admin')
```

</TabItem>
<TabItem value="lua" label="Lua">

```lua title="/rc/server/player/player.class.ts"
local NX = exports['NX']:useServer()

local nxPlayer = NX.Players.Get(source)

nxPlayer.SetPermissions('admin')
```

</TabItem>
</Tabs>

---

| Argument   | Data Type | Optional | Default Value | Explanation             |
| ---------- | --------- | -------- | ------------- | ----------------------- |
| permission | String    | No       | -             | The permission to set.. |
