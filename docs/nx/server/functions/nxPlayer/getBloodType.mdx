# GetBloodType

:::info
This function will return the player blood type
:::

> `'A+' | 'A-' | 'B+' | 'B-' | 'AB+' | 'AB-' | 'O+' | 'O-'`

```typescript
nxPlayer.GetBloodType()
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript/JavaScript">

```typescript title="/rc/server/player/player.class.ts"
const NX = exports.NX.useServer()

const nxPlayer = NX.Players.Get(source)

console.log(nxPlayer.GetBloodType())
```
