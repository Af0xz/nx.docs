# Setting up

:::caution
**make sure you have the following dependencies installed.**
:::

- [Node.JS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/lang/en/docs/install/)
- [Git](https://git-scm.com/)

## Installing dependencies

**Step 1**

clone the repository into your `resources` folder

```sh
git clone https://github.com/niiyy/NX NX
```

**Step 2**

navigate to `/rc` and run

```sh
yarn
```

**Step 3**

navigate to `/ui` and run

```sh
yarn
```

## Building files

**Building NUI**

navigate to `/ui` and run.

```sh
yarn build
```

**Building Game Scripts**

navigate to `/rc` and run

```sh
yarn build
```

## Watching files

**Watvhing NUI**

to watch nui files in FiveM env you need to navigate to `/ui` and run

```sh
yarn dev:ig
```

to watch nui files in browser env you need to navigate to `/ui` and run

```sh
yarn dev
```

**Watvhing Game Scripts**

to watch game files you need to navigate to `/rc` and run

```sh
yarn watch
```

<br />

:::danger
the command `yarn build` in main root is used to create a new version do not use it
:::
