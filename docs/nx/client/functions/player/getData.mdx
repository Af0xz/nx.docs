# GetData

This function well return the player data.

```typescript
NX.Player.GetData()
```

---

## Example

<Tabs>


</TabItem>

## Returns

```js
{
  accounts,
    position,
    identifier,
    inventory,
    charinfo,
    permissions,
    skin,
    uid,
    ped
}
```
