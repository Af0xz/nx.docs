# Destroy

This function well destroy the actual input menu.

```typescript
NX.Input.Destroy(cb)
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript">

```typescript
const NX = exports.NX.useClient()

NX.Input.Destroy((res: RespT) => {
  if (res.status === 'succes') {
    console.log('Destroyed !')
  }
})

interface RespT {
  status: 'succes' | 'error'
  message?: any
  data?: any
}
```

</TabItem>
