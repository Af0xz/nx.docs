# Create

This function well create an input menu.

```typescript
NX.Input.Create(data, cb)
```

---

## Example

:::caution

**ID** most be **unique**.

:::

<Tabs>

        label: 'Input 1',

  },

    //

## Arguments

| Argument | Data Type | Optional | Default Value | Explanation       |
| -------- | --------- | -------- | ------------- | ----------------- |
| Data     | Object    | No       | -             | The configuration |
| Callback | Function  | No       | -             | Message subject   |

#### Data

| Argument | Data Type | Optional | Default Value | Explanation              |
| -------- | --------- | -------- | ------------- | ------------------------ |
| title    | String    | No       | `title`       | The title of the menu.   |
| rows     | Array     | No       | -             | The array of the inputs. |

#### Input

| Argument | Data Type | Optional | Default Value | Explanation                       |
| -------- | --------- | -------- | ------------- | --------------------------------- |
| label    | String    | No       | -             | The label of the input.           |
| id       | String    | No       | -             | The id of input `MOST BE UNIQUE`. |
