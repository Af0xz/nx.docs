# Create

This function well update the timeline.

```typescript
NX.Timeline.Update(data)
```

---

## Arguments

| Argument | Data Type | Optional | Default Value | Explanation       |
| -------- | --------- | -------- | ------------- | ----------------- |
| data     | Object    | No       | -             | The configuration |

### Rows

| Argument | Data Type | Optional | Default Value | Explanation                                                   |
| -------- | --------- | -------- | ------------- | ------------------------------------------------------------- |
| type     | String    | No       | -             | The update type can be: `SET_COMPLETED` or `SET_UNCOMPLETED`. |
| id       | String    | No       | -             | The id of the row to update                                   |
