# Create

This function well repair the vehicle provided or the vehicle ped is in.

```typescript
NX.Vehicles.Repair(vehicle)
```

## Arguments

| Argument | Data Type | Optional | Default Value    | Explanation           |
| -------- | --------- | -------- | ---------------- | --------------------- |
| vehicle  | Number    | Yes      | `VehiclePedIsIn` | The vehicle to repair |
