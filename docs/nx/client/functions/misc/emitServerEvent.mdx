# EmitServerEvent

This function well emit a server event.

```typescript
NX.Misc.EmitServerEvent(eventName, cb, ...args)
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript">

```typescript title="/rc/client/class/misc.ts"
const NX = exports.NX.useClient()

NX.Misc.EmitServerEvent(
  'NX::helloWorld',
  (name: string, amount: number) => {
    console.log(name)
    console.log(amount)
  },
  'arg1',
  'arg2'
)
```

</TabItem>

## Arguments

| Argument  | Data Type | Optional | Default Value | Explanation                                              |
| --------- | --------- | -------- | ------------- | -------------------------------------------------------- |
| eventName | String    | No       | -             | The event name to trigger                                |
| cb        | Function  | No       | -             | The callback that well be executed after server response |
| args      | Any       | Yes      | -             | The args to send to server side                          |
