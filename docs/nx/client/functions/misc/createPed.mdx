# CreatePed

This function well create a new ped.

```typescript
NX.Misc.CreatePed(pedType, model, cb)
```

---

  //

  status: 'succes' | 'error'

  data?: any

## Arguments

| Argument | Data Type | Optional | Default Value | Explanation                                                                  |
| -------- | --------- | -------- | ------------- | ---------------------------------------------------------------------------- |
| pedType  | Number    | No       | -             | Can be `1` or `2`                                                            |
| model    | String    | No       | -             | The ped model [see](https://docs.fivem.net/docs/game-references/ped-models/) |
| cb       | Function  | Yes      | -             | The callback [see](https://docs.fivem.net/docs/game-references/ped-models/)  |
