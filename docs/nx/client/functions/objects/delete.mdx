## Arguments

| Argument | Data Type | Optional | Default Value | Explanation                                                 |
| -------- | --------- | -------- | ------------- | ----------------------------------------------------------- |
| entity   | Number    | No       | -             | The object ID                                               |
| cb       | Function  | Yes      | -             | The callback that well be executed after object has deleted |
