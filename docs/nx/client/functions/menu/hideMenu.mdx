# HideMenu

This function will hide the menu to player.

```typescript
yourMenuName.HideMenu()
```

---

## Example

<Tabs>
<TabItem value="ts" label="TypeScript">

```typescript title="/rc/client/menu/menu.service.ts"
const NX = exports.NX.useClient()

const yourMenuName = NX.Menu.CreateMenu({
  options: {
    title: 'Custom',
    banner:
      'https://img.freepik.com/free-vector/hand-painted-watercolor-pastel-sky-background_23-2148902771.jpg?w=2000',
    width: 400,
  },
  items: [
    {
      type: 'BUTTON',
      label: 'Valider',
      id: 'btn-1',
    },
  ],
})

yourMenuName.HideMenu()
```

</TabItem>
