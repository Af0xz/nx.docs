# Create

This function well create an input menu.

```typescript
NX.Menu.CreateMenu({ options, items })
```

---

## Example

:::caution

**ID** most be **unique**.

:::

## Arguments

| Argument | Data Type | Optional | Default Value | Explanation                    |
| -------- | --------- | -------- | ------------- | ------------------------------ |
| Options  | Object    | No       | -             | The configuration for the menu |
| Items    | Object    | No       | -             | What the menu contains         |

### Options

| Argument | Data Type | Optional | Default Value | Explanation                   |
| -------- | --------- | -------- | ------------- | ----------------------------- |
| title    | String    | No       | -             | The title of the menu.        |
| banner   | URL       | No       | -             | The background Color / image. |
| width    | number    | Yes      | `400`         | The Menu Size.                |

### Items

| Argument | Data Type | Optional | Default Value | Explanation                                                                  |
| -------- | --------- | -------- | ------------- | ---------------------------------------------------------------------------- |
| type     | String    | No       | -             | The type of the item can be `BUTTON & SLIDER & LIST & CHECKBOX & SEPARATOR`. |
| label    | String    | No       | -             | The label of the item.                                                       |
| id       | String    | No       | -             | The id of Item `MOST BE UNIQUE`.                                             |
| onClick  | Function  | Yes      | -             | Detects item is clicked `ONLY FOR BUTTON`.                                   |
| onChange | Function  | Yes      | -             | Detects when item is change `ONLY FOR LIST & CHECKBOX`.                      |
